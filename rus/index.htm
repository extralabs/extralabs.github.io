Feed Editor

Если вы хотите создавать профессиональные RSS ленты, но у вас катастрофически нет времени на изучение технологии RSS, тогда Feed Editor - решение для вас! Feed Editor генерирует ленты, используя версию 2.0 стандарта RSS, и является лучшим решением для WYSIWYG редактирования RSS каналов и подкастов, не требуя при этом знания основ XML.

 

Feed Mix

Feed Mix - уникальный инструмент для создания RSS. С помощью этой программы вы можете создавать новый RSS ленты из нескольких уже существующих, всего за пару минут! Просто укажите, какие новости из выбранных RSS каналов вы бы хотели видеть в своей новой RSS ленте и опубликуйте готовый RSS на вашем сайте!

 

RSS Wizard

RSS Wizard - это программа для конвертирования HTML в RSS, которая может генерировать RSS из любой Web-страницы, не требуя предварительного редактирования ее кода. Программа имеет встроенный планировщик, XML редактор, и способна работать в полностью автоматическом режиме.


 

Feed2Html PHP Script

Feed2Html - это бесплатный PHP скрипт, который конвертирует данные из RSS в HTML, используя темплейты для дизайна конечных страниц. Если вы уже имеете RSS на вашем сайте, просто установите этот скрипт. Скрипт обновляет Web-страницы "на лету", делая доступной информацию из ваших RSS для поисковых роботов.
