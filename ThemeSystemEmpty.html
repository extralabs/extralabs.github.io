Feed Mix: Quick Reference
Please read this step-by-step example:
Full version available here

1. Create the new RSS Feed "Project". To do that click on the button "New Project" on the Project button bar (to the right).

2. Select the source RSS feeds (can use this catalog), and add them in Feed Mix, for this press on the button "Open Existing Feed" on the Feed button bar (to the left).

3. Every time you add a new feed, it will be loaded automatically. If it hasn't happened, then press "". After a new feed is loaded its contents will be displayed in the View area (here).

4. Select and add the feed information (e.g. topics, image) from the "Feed" to the "Project", by using View area button bar, and "".

5. If necessary, edit the layout of the "Project" Feed by using the Data entry area (to the right).

6. Upload the created RSS Feed "Project" to your Web-site using the built-in FTP tool (by clicking the "FTP upload" button on the Project button bar).
