Automatic Email Newsletters with Easy RSS to Email!

Easy RSS-to-Email is an application that extracts content from web feeds, creates email messages, and sends them. It supports RSS, Atom, and RDF formats, and can generate HTML or plain-text messages.

There is a built-in dictionary for automatic synonymizing, a keyword filter for feed topics, and a mixer for web feeds. Thanks to the excellent support of all major text databases, you can easily export or import email addresses.

Our application has a simple graphical interface and is easy to use even for a novice. We also provide a command line interface for more experienced users.

To create a newsletter, click "New Project" on the application's toolbar. When the New Project dialog box opens, enter web feeds addresses and select a built-in email template. Then manually enter or import your recipients' email addresses, and click "OK."

Clicking the "Start" button on the application's toolbar will start the emailing. Each web feed will be loaded and processed, and generated messages will be sent.
