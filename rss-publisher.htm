RSS Publisher

Who can use this application?
RSS Publisher is intended for people who frequently update web feeds on their sites.

What's the purpose of this application?
If you use web feeds, you probably have to update them regularly. Alas, when you upload a new feed to your server, the old one is overwritten or effectively lost. Our application solves this data-loss problem! It fully automates web-feed uploading and updating, and also lets you archive old web feeds in a database that contains feeds and related files.

How does this application work?
First create a new task and enter all necessary parameters, including your web server address, login, password, and the location of RSS files. After that, you can leave updating and archiving of your web feeds to our application.

RSS Publisher has a unique feature called "Smart Backup," which automatically compares the existing RSS file on your web server with your local RSS file. If these files are different in some respect (as specified by you), first the remote file will be backed up on your local computer, and then the local file will be uploaded to the server.
