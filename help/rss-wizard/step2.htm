Step 2 of 4: Generation parameters

 

To generate an RSS file, RSS Wizard uses the data contained in the loaded web page. For more information on the structure of RSS files, click here. The converter functions in the following way: it picks out the necessary texts (as set by the user) in the web page code and uses them to create an RSS feed.


RSS Wizard has a unique feature: it can automatically determine the Description, Title and Link of each item. However, if for some reason you do not want to use it, specify the parameters of these fields manually.

Besides, you don't have to fill in each field of Step 2 manually. Press the "Select" button to invoke the window that will display the code of the loaded page. Select the necessary part of the code and press "OK".

Start String
The identifier that marks the beginning of the code that will be processed.

Start Item String
Specifies the beginning of each RSS feed item.

End Item String
Specifies the end of each item.

End String
Specifies the end of the processed part of the code.

Start Description String
Specifies the beginning of each RSS feed item description.

End Description String
Specifies the end of each RSS feed item description.

Start Title String
Specifies the beginning of each RSS feed item title.

End Title String
Specifies the end of each RSS feed item title.

Start Link String
Specifies the beginning of each RSS feed item link.

End Link String
Specifies the end of each RSS feed item link.
