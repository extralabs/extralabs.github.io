Step 1 of 4: Selecting file

 


Source File
Specify the link to the file or press the "Browse" button to select a local file. Use the "http://" prefix for online documents and "file://" for local files.

When you press the "Next" button, the file will be loaded and the wizard will proceed to Step 2.
