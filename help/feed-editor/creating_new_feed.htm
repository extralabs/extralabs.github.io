Creating New Feed

Top  Previous  Next

 

 

To get a new feed, you can use one of the following:

 

· Create a blank feed. Click the “New Feed” button in the Feed toolbar or select the “New…” item of the “Feed” menu. You can also use the “Ctrl+N” key combination.
· Use the New Feed Creation Wizard. Click the corresponding button in the Feed toolbar or select the “Wizard…” item of the “Feed” menu.

 

The created RSS feed becomes current and can be immediately edited, published etc.
