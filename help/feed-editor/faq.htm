FAQ

What is Feed Editor?

Feed Editor is a unique powerful and easy-to-use tool for creating RSS feeds and podcasts.

 

I'm very new to RSS. How difficult is it to create your own RSS feed?

With Feed Editor, it is really easy to create new feeds and update already existing RSS feeds. Watch the video tutorial to start with.

 

Where can I learn more about RSS fields, for example, about the "GUID" field?

Feed Editor has a built-in context help system. To get help, select the “Help\What’s This? Help” menu item and click any field in the Feed Editor window.

 

Where can I get additional information about RSS technologies and podcasting?

We have created several special information resources devoted to RSS:

http://www.rss-specification.com/

http://www.create-rss-feed.com/

http://www.rss-software.org/

 

I have created an RSS feed, but I'm not sure if it is valid. How can I validate the feed?

Use the built-in Feed Validator (“Tools\Feed Validator…”) to check whether your feeds and podcasts comply with the RSS and iTunes specifications.

 

How to publish an RSS feed?

There are several ways to publish an RSS feed. Method 1. Use the control panel of your hosting service provider, manually upload the RSS feed file (.xml or .rss) to your server. Method 2. If you hosting service provider supports transferring files via FTP (File Transfer Protocol), you can use the built-in FTP client (“Feed\Upload…”).

 

I have successfully uploaded my RSS feed to the server. How to do so that the browser could automatically detect the RSS feed (for example, the RSS icon to the left from the page URL in Internet Exporer)?

You should create a link to the RSS feed in the “<head>” section of each page on your site:

<head> 
<link rel="alternate" type="application/rss+xml" title="RSS" href="http://www.yoursite.com/yourfeed.xml" /> 
</head>

 

How can I increase the popularity of my RSS feed?

The main thing you should do is submit your feed to RSS directories and search engines. There are hundreds of these sites available at the moment. Use RSS Submit to automatically submit your feed to these sites.

 

Is there any simple way to move items in a feed?

There are several methods you can use.

1). You can use the “Move Item Up”/”Move Item Down” buttons;

2). Move items by just dragging them while holding down the left mouse button over the list with feed items;

3). Use the clipboard to copy and paste whole feed items.

 

How can I convert MySQL/CSV into RSS?

Use the Import Wizard (“Tools\Import Wizard…”). You can convert HTML, CSV into RSS. Besides, you can use Feed Editor to export data from RSS to CSV, Excel, Atom, RDF, JS, HTML, TXT.

 

I still have questions. Where can I get help?

Contact the Extralabs Software technical support service. We will be glad to help you. Questions from registered users have a higher priority and are answered first.
